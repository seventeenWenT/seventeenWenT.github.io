安装步骤


这里使用yum安装，但是必须先要替换yum源。
首先修改yum源

yum install yum install http://mirrors.sohu.com/fedora-epel/6/x86_64/epel-release-6-8.noarch.

直接执行yum安装Absible

yum install Ansible -y

安装完成后检查一下：

rpm -qa|grep "ansible"
